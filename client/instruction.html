Chord Transposer expects a document like the following.

    A      B       C
Big wheels keep on turning
      D       E      F
Carry me home to see my kin

You then tell it the current key and which key to transpose to. If you select from key "A" to "B" and click Transpose, the document will become:

    B      C#      D
Big wheels keep on turning
      E       F#     G
Carry me home to see my kin

By default it will transpose the entire document. If you only want to transpose part of the document, just select that portion before you transpose.
